# Claude Code란 무엇인가?

## 한 줄 요약

Claude Code는 터미널(명령줄)에서 AI와 대화하며 작업을 수행하는 도구입니다.

## 누가 사용할 수 있나요?

Claude Code는 개발자뿐만 아니라 다양한 직군에서 활용할 수 있습니다:

- 개발자: 코드 작성, 리뷰, 디버깅, 리팩토링
- PM/기획자: 기획서 작성, 이슈 관리, 데이터 정리
- 디자이너: CSS 작업, 디자인 시스템 관리, 접근성 검사
- 리서처: 데이터 수집/분석, 리포트 생성
- 콘텐츠 크리에이터: 블로그 작성, SEO 최적화
- 학생: 코딩 학습, 프로젝트 실습

## 기존 AI 챗봇과 뭐가 다른가요?

|            | 웹 AI 챗봇  | Claude Code                         |
| ---------- | ----------- | ----------------------------------- |
| 인터페이스 | 웹 브라우저 | 터미널 (CLI)                        |
| 파일 접근  | 불가능      | 내 컴퓨터의 파일을 직접 읽고 수정   |
| 명령 실행  | 불가능      | 터미널 명령어 직접 실행             |
| 자동화     | 제한적      | 커맨드, 스킬, 에이전트로 자동화     |
| 맞춤 설정  | 불가능      | CLAUDE.md, rules, hooks로 맞춤 설정 |

## 핵심 개념 5가지

### 1. CLAUDE.md (설정 파일)

Claude Code에게 "나는 이런 사람이고, 이렇게 일해줘"라고 알려주는 파일입니다.
위치: `~/.claude/CLAUDE.md` (전역) 또는 프로젝트 루트의 `CLAUDE.md`

### 2. 커맨드 (Commands)

`good-vibe:hello`, `good-vibe:learn` 처럼 플러그인 접두사로 시작하는 단축 명령어입니다.
자주 하는 작업을 한 번의 입력으로 실행할 수 있습니다.

### 3. 스킬 (Skills)

특정 작업을 수행하는 방법을 정의한 가이드입니다.
예: TDD 워크플로우, 코드 리뷰 체크리스트, SEO 최적화 등

### 4. 에이전트 (Agents)

특정 역할을 수행하는 AI 도우미입니다.
예: 코드 리뷰어, 문서 검토기, 접근성 체커 등

### 5. 훅 (Hooks)

특정 이벤트가 발생했을 때 자동으로 실행되는 작업입니다.
예: 파일 저장 시 자동 포맷팅, 커밋 시 린트 검사 등

## 시작하기

Claude Code를 설치했다면, 터미널에서 `claude`를 입력하세요:

```bash
claude
```

대화가 시작되면 자연스럽게 한국어로 요청하면 됩니다:

```
> 이 폴더에 있는 파일 목록을 보여줘
> README.md 파일을 읽어줘
> package.json에 vitest를 추가해줘
```
