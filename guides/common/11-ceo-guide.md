# CEO 가이드 — 비개발자를 위한 Good Vibe Coding 안내

## Good Vibe Coding이란?

Good Vibe Coding은 **AI 팀을 만들어서 프로젝트를 진행하는 도구**입니다.
개발자가 아니어도, 아이디어만 있으면 AI 팀이 기획부터 구현까지 처리합니다.

당신의 역할은 **CEO** — 방향을 정하고, 중요한 판단을 내리는 것입니다.

## 핵심 용어 해설

| 용어             | 뜻                                                               |
| ---------------- | ---------------------------------------------------------------- |
| **Phase**        | 작업 단위. 큰 프로젝트는 여러 Phase로 나뉘어 순서대로 진행됩니다 |
| **팀 검토**      | 한 팀원이 만든 결과물을 다른 팀원이 검토하는 과정                |
| **품질 검증**    | 심각한 문제가 없는지 자동으로 확인하는 단계                      |
| **에스컬레이션** | AI 팀이 자체 해결이 안 될 때, CEO에게 판단을 요청하는 것         |
| **토론**         | AI 팀원들이 기획 방향을 논의하는 과정                            |
| **분석그룹**     | 비슷한 역할의 팀원들이 묶인 그룹 (전략팀, 구현팀, 검증팀 등)     |

## CEO의 역할

### 당신이 하는 것

1. **아이디어 제시** — "이런 걸 만들어줘"
2. **승인** — AI 팀의 기획안을 검토하고 승인
3. **판단** — 문제가 생겼을 때 방향 결정

### 당신이 안 해도 되는 것

- 코드 작성
- 기술 결정
- 팀원 관리

## 판단이 필요한 순간

### 1. 프로젝트 모드 선택

`good-vibe:new`에서 자동으로 추천하지만, 직접 선택할 수도 있습니다:

- **간단한 프로젝트** — AI 팀이 바로 만듭니다 (봇, 스크립트 등)
- **중간 규모** — 한 번 토론 후 바로 실행합니다 (웹앱, API 등)
- **대규모 프로젝트** — 충분히 토론하고, CEO가 승인한 후 실행합니다

### 2. 실행 모드 선택

- **인터랙티브** — Phase마다 진행 여부를 확인합니다 (추천)
- **배치 실행** — 3개 Phase마다 확인합니다
- **자동 실행** — 문제가 생길 때만 멈춥니다

### 3. 에스컬레이션 판단

AI 팀이 2번 수정을 시도했는데도 문제가 해결되지 않으면, 당신에게 물어봅니다:

- **계속 수정** — "이건 꼭 필요한 기능이야, 한 번만 더 해봐"
- **건너뛰기** — "이건 나중에 해도 돼, 다음으로 넘어가"
- **중단** — "처음부터 다시 생각해보자"

> 잘 모르겠으면? **건너뛰기**가 안전합니다. 나중에 `good-vibe:modify`로 추가할 수 있습니다.

## 기본 워크플로우

```
1. good-vibe:hello    → 환경 설정 (처음 1회)
2. good-vibe:new      → 프로젝트 시작
3. good-vibe:discuss  → AI 팀 토론 (자동 진행)
4. good-vibe:approve  → 기획안 승인
5. good-vibe:execute  → 작업 실행
6. good-vibe:report   → 결과 보고서
```

## 완성 후 다음 단계

프로젝트가 완료되면:

1. **보고서 확인** — `good-vibe:report`로 어떤 것이 만들어졌는지 확인
2. **피드백** — `good-vibe:feedback`으로 팀원 성과 분석
3. **기능 추가/수정** — `good-vibe:modify`로 기존 프로젝트에 변경 사항 추가

## FAQ

**Q: 기술 용어를 몰라도 되나요?**
A: 네. AI 팀이 기술 결정을 하고, CEO에게는 이해하기 쉬운 요약만 보여줍니다.

**Q: 도중에 방향을 바꿀 수 있나요?**
A: 네. 인터랙티브 모드에서 Phase마다 지침을 추가하거나 중단할 수 있습니다.

**Q: 에스컬레이션에서 뭘 선택해야 할지 모르겠어요.**
A: "건너뛰기"를 선택하면 해당 부분을 넘기고 나머지를 완성합니다. 나중에 추가할 수 있으니 안전한 선택입니다.

**Q: 완성된 결과물은 어떻게 쓰나요?**
A: `good-vibe:report` 보고서에 실행 방법이 포함됩니다. 환경변수 설정, 의존성 설치, 실행 명령어 등이 안내됩니다.
