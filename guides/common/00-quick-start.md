# 3분 퀵스타트

Good Vibe Coding은 AI 팀을 구성해서 프로젝트를 함께 만드는 도구입니다.
당신이 CEO가 되어 아이디어를 말하면, AI 팀원들이 기획하고 만들어줍니다.

---

## 6단계로 프로젝트 완성하기

### Step 1: `/hello` — 프로젝트 인프라 셋업

**무슨 일이 일어나나요?**
프로젝트 폴더, CLAUDE.md, README.md를 만들고 GitHub 저장소를 생성합니다.

**소요시간:** 2-3분

**예시 입력:**
```
/hello
→ 프로젝트 이름: "weather-bot"
→ 설명: "날씨를 알려주는 텔레그램 봇"
→ 기술 스택: Next.js
→ GitHub: 비공개 저장소
```

---

### Step 2: `/new` — 아이디어 입력 + 팀 자동 구성

**무슨 일이 일어나나요?**
프로젝트 아이디어를 입력하면, 복잡도를 자동 분석하고 적합한 팀을 구성합니다.

**소요시간:** 1-2분

**예시 입력:**
```
/new
→ "날씨를 알려주는 텔레그램 봇을 만들고 싶어"
```

---

### Step 3: `/discuss` — 팀 토론 → 기획서 작성

**무슨 일이 일어나나요?**
AI 팀원들이 각자 프로젝트를 분석하고, 결과를 모아 기획서를 만듭니다.

**소요시간:** 3-10분 (프로젝트 규모에 따라)

**예시 입력:**
```
/discuss
```

---

### Step 4: `/approve` — 기획서 확인 + 승인

**무슨 일이 일어나나요?**
완성된 기획서를 보여주고, 승인/수정/재토론을 선택할 수 있습니다.

**소요시간:** 1-2분

**예시 입력:**
```
/approve
→ "승인" 선택
```

---

### Step 5: `/execute` — 작업 실행 + 리뷰

**무슨 일이 일어나나요?**
팀원들이 분배된 작업을 실행하고, 다른 팀원이 결과물을 리뷰합니다.

**소요시간:** 5-15분 (작업량에 따라)

**예시 입력:**
```
/execute
```

---

### Step 6: `/report` — 보고서 생성

**무슨 일이 일어나나요?**
프로젝트 전체 과정을 정리한 보고서를 생성합니다.

**소요시간:** 1분

**예시 입력:**
```
/report
```

---

## 보조 커맨드 (필요할 때 사용)

| 커맨드 | 설명 |
|--------|------|
| `/status` | 프로젝트 진행 상태 확인 |
| `/feedback` | 팀원 성과 분석 + 개선 제안 |
| `/my-team` | 팀원 정보 보기 |
| `/learn` | 더 자세한 사용법 학습 |
